2. **Open any code file**
   - Works with JavaScript, TypeScript, Python, Java, and more

3. **Right-click or use Command Palette**
   - Right-click in the editor and select **"Review Code with AI Mentor"**
   - Or press `Ctrl+Shift+P` and run **"AI Mentor: Review Current File"**

4. **View Results**
   - See AI-powered feedback, bug detection, security analysis, and improvement suggestions in a sidebar panel or output channel
